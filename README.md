# 這是一個每次喚醒電腦會"提醒你今天的行程"的程式
每日事項記在 excel 資料夾內的 Calender.xlsx 檔案中

## 安裝第三方模組 openpyxl
```pip install openpyxl```\
需先安裝 openpyxl 才可執行 everyday.py 和 create.py
